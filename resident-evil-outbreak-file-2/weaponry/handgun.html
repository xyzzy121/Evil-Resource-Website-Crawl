Handgun (Resident Evil Outbreak File #2)

The RPD's official handgun. It uses 9mm parabellum rounds.

A lightweight automatic pistol. It fires 9mm parabellum rounds.

Its ammunition is Handgun Rounds and it is also compatible with the Handgun Magazine.
