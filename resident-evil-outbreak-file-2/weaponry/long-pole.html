Long Pole (Resident Evil Outbreak File #2)

A relatively long wooden pole. It is relatively rigid.

A long pole made from wood. You can use this as a weapon that is capable of causing a small amount of damage.

This weapon can only be created by David. It is created by combining two Wooden Poles together using a piece of Vinyl Tape.

It is also possible for David to combine this with a Butcher Knife using a piece of Vinyl Tape to create a Spear.
CategoryWeaponry (Melee weapon)
