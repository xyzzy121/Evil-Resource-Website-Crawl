Pesticide Spray (Resident Evil Outbreak File #2)

An aerosol spray. It contains pesticide.

An aerosol can containing insect extermination gas. It can be wielded as a weapon, and when sprayed in an enemies face it will cause them to stagger backwards.

David can convert this into a more deadly weapon by combining this with a Lighter using a piece of Vinyl Tape. This will create the Flame Spray.
CategoryWeaponry (Melee weapon)
