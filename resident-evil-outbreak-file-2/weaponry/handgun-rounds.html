Handgun Rounds (Resident Evil Outbreak File #2)

9mm parabellum rounds. They are used in various handguns.

A box of 9mm parabellum rounds.

These bullets can be used with the following weapons:
  • Handgun
  • Handgun for Mark
  • Handgun HP
  • Handgun GL
  • Handgun SG
  • Burst Handgun
  • Sub Machinegun

They can also be used to refill the following magazines:
  • Handgun Magazine
  • Handgun HP Magazine
  • Handgun GL Magazine
  • Handgun SG Magazine
  • Burst Handgun Magazine
  • Submachinegun Magazine
