Employee Area Key (Resident Evil Outbreak File #2)

The tags reads: "Employee use only."

A small red key with a yellow tag attached.

This key is used within the "Underbelly" scenario. It can be used to unlock the doors in either the East concourse or the West concourse to gain access to the Employee area - Subway B1F.
