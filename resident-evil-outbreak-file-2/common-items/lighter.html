Lighter (Resident Evil Outbreak File #2)

J's BAR is inscribed on it.

A standard flip-top lighter.

This item has several useful and important uses:

  • In the "Flashback" scenario, when playing on Very Hard mode, you need to use the lighter in the main building 2F hall to find the Rusty Key. You can also the use the lighter in the intensive care unit and the maintenance access route to find more hidden items.
  • You can create a Molotov Cocktail by combing the lighter with a Newspaper + Bottle.
  • David can combine the lighter with a can of Pesticide Spray to create Flame Spray, provided he has at least one strip of Vinyl Tape left.
