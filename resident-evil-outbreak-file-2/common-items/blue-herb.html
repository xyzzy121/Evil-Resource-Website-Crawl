Blue Herb (Resident Evil Outbreak File #2)

An herb which grows wild in this area.

One of three different herbs found growing in pots in and around Raccoon City.

This herb can be used to neutralize the effect of any poison, making it very useful after encountering venomous enemies. It can also be combined with other herbs to create more powerful medicines.

  • Combine with a Green Herb to create Mixed Herb (G + B).
  • Combine with a Red Herb to create Mixed Herb (B + R).
  • George can combine this with his Medical Set to create Recovery Medicine.
CategoryCommon item (Recovery item)
