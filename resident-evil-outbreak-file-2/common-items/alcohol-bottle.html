Alcohol Bottle (Resident Evil Outbreak File #2)

A bottle with a flammable alcohol in it.

An ordinary glass bottle used for storing alcohol.

This bottle has no use on its own, but it can be combined with one of the various newspapers you find to create the Newspaper + Bottle, which in turn can be combined with the Lighter to create a Molotov Cocktail.
