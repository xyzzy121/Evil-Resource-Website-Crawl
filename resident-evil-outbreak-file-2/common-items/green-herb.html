Green Herb (Resident Evil Outbreak File #2)

An herb which grows wild in this area.

One of three different herbs found growing in pots in and around Raccoon City.

The most useful of the three herbs you can find. The herb by itself will restore a small amount of health, but can be enhanced by combining it with other herbs to create new medicines.

  • Combine with a Red Herb to create Mixed Herb (G + R).
  • Combine with a Blue Herb to create Mixed Herb (G + B).
  • Combine with another Green Herb to create Mixed Herb (G + G).
  • George can combine this with his Medical Set to create an Antidote.
CategoryCommon item (Recovery item)
