Newspaper 1 (Resident Evil Outbreak File #2)

Just an ordinary newspaper. (Check FILE for contents.)

One of three types of newspaper you find that contains an interesting an article.

By acquiring this you will automatically have a file added to your inventory. The following files will be obtained:
  • In "Underbelly" you will receive the Raccoon Today file.
  • In "End of the Road" you will receive the Raccoon Today file.

You can also use this newspaper to create a Molotov Cocktail by combining it with an Alcohol Bottle and then with a Lighter.
