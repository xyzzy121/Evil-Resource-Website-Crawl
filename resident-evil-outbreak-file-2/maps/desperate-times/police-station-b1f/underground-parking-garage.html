Underground parking garage - Police station B1F (Resident Evil Outbreak File #2)

The shutter in the north west corner leading to the parking garage ramp can only be opened from the Main Hall computer, which only becomes available for use once you have placed all five of the gemstone plates into the fountain.

This location also contains a shortcut. On the south side of the garage there are a stack of crates behind a parked car. Moving the first crate will reveal a crawlspace that leads to the police station front entrance.

This place is linked to the following locations: B1F East hall, B1F West hall, Parking garage ramp, Police station front entrance.
