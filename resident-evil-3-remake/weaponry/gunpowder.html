Gunpowder (Resident Evil 3 Remake)

Standard-grade gunpowder. Combine 2 of these to create handgun ammo.

A small container of gunpowder used to create ammunition.

You can use this in the following combinations:
  • Combine with another Gunpowder to create Handgun Ammo.
  • Combine with High-Grade Gunpowder to create Shotgun Shells.
