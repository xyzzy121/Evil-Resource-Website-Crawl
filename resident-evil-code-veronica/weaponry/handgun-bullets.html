Handgun Bullets (Resident Evil Code: Veronica)

9mm x 19 Rounds. These can be used with the M93R and Glock 17.

A box of fifteen 9mm parabellum rounds.

These bullets can be used with the following weapons:
  • M93R
  • M93R Burst
  • Glock 17
  • Modified Glock 17
