Note from Conductor (Resident Evil Zero)

A note left by the conductor of the train describing where to find the Blue Keycard.
CategoryFile

Transcript

Note from Conductor

The key card for the Driver's Compartment is in my bag as always.

But I need the other guy's key too to open the damn thing.

But, it looks like I've lost it somewhere.

If it was a normal key, someone would be sure to turn it in, but this key is different...

I'll look for it too, but if you happen to find it, please hand it in.

Thanks in advance.
