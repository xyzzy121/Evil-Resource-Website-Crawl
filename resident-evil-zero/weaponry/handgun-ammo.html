Handgun Ammo (Resident Evil Zero)

Holds 9x19 parabellum rounds for use with handguns.

A box of fifteen 9mm parabellum rounds.

These bullets can be used with the following weapons:
  • Rebecca's Handgun
  • Billy's Handgun
  • Rebecca's Custom Handgun
  • Billy's Custom Handgun
