Dirty Pearl Pendant (Resident Evil 4)

A very dirty accessory. It doesn't seem to have much value.

A dirty pearl pendant that doesn't seem to have much value. You can sell this to the merchant to make some extra Pesetas. However, if you closed the lid above the well before shooting this down, it will stop it getting dirty, and the value will increase.
CategoryTreasure
Sell price1,000 pesetas

Location

Normal ModeProfessional Mode

  • Chapter 1-1

    Normal Mode
    Farm. You retrieve this from the dirty well if you shot down the Pearl Pendant without first closing the lid.

  • Chapter 1-1

    Professional Mode
    Farm. You retrieve this from the dirty well if you shot down the Pearl Pendant without first closing the lid.
