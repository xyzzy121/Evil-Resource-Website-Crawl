Authentication Code (Resident Evil Revelations)

USB flash drive containing passcode for laboratory security system.

A USB token that contains the code for destroying the stored virus on board the Queen Zenobia.

Once you have acquired this you need to use it on the central console on the Main Platform of the Laboratory.
