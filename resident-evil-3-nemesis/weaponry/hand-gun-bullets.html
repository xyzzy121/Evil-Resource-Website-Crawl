Hand Gun Bullets (Resident Evil 3: Nemesis)

9x19 parabellum rounds. Used for either the M92FS Custom or the SIGPRO.

A box of thirty 9mm parabellum rounds for use with the following weapons:
  • M92F Custom
  • SIGPRO SP2009
  • STI EAGLE 6.0

These also can be created by using various combinations of Gun Powder A's with the Reloading Tool.
