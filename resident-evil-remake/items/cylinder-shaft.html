Cylinder Shaft (Resident Evil Remake)

Symbols are carved in its side.

A brass metal object with symbols printed around the shaft.

This is what's required to restore power to the underground elevator in the Generator Room. It is created by combining the Cylinder with the Shaft. Once done you can insert it into the machine, but you will find you need to enter a code in order to finish the process. The code can be found on the cylinder shaft itself, and when you examine it you can rotate its top half to match up the symbols along its side. Eventually with enough rotations the Roman numerals "IV II III I" will be displayed, which translates to "4231".
